## ADDED Requirements

### Requirement: 演示指南文档
`docs/demo.md` SHALL 提供完整的项目演示指南，包含默认账号、管理端演示路径和用户端演示路径。

#### Scenario: 演示者快速准备演示环境
- **WHEN** 演示者打开 docs/demo.md
- **THEN** 看到默认管理员账号和密码（admin/admin123）
- **AND** 看到普通用户注册步骤（用于用户端演示）
- **AND** 看到系统启动确认步骤

#### Scenario: 演示管理端功能
- **WHEN** 演示者查看管理端演示路径
- **THEN** 看到完整的操作步骤：登录 → 商品管理 → 订单管理 → 优惠券管理 → 评论审核
- **AND** 每个步骤说明要展示的核心功能和预期效果

#### Scenario: 演示用户端交易链路
- **WHEN** 演示者查看用户端演示路径
- **THEN** 看到完整的购物流程：注册/登录 → 浏览商品 → 加入购物车 → 使用优惠券 → 创建订单 → 模拟支付 → 查看订单状态
- **AND** 看到会员功能演示：签到 → 查看积分 → 积分兑换 → 查看会员等级
- **AND** 看到社交功能演示：商品评论 → 收藏商品

#### Scenario: 演示核心交易链路设计点
- **WHEN** 演示者查看"交易链路亮点"部分
- **THEN** 看到如何演示库存锁定（创建订单后检查库存变化）
- **AND** 看到如何演示订单超时自动取消（创建订单后等待超时）
- **AND** 看到如何演示支付幂等（重复点击支付）
- **AND** 看到如何演示优惠券锁定与返还（创建订单后取消）
